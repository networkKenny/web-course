1.元素選擇器
    可以選擇在html檔案中「所有」符合該名稱的標籤，並對其下設定
如何使用：
    如果你要選擇所有的a標籤
    a{
        一些設定
    }


2.Class選擇器
    可以選擇在html檔案中「所有」符合該class名稱的標籤，並對其下設定
如何使用：
    如果你要選擇所有class名稱是my-class的標籤
    .my-class{
        一些設定
    }


3.ID選擇器
    可以選擇在html檔案中「所有」符合該id名稱的標籤，並對其下設定
如何使用：
    如果你要選擇所有id名稱是my_id的標籤
    #my_id{
        一些設定
    }


混合式選擇器：
    如果你要選擇所有的a和p標籤
    a, p{
        一些設定
    }
    如果你要選擇所有class名稱是my-p的p標籤
    p.my-p{
        一些設定
    }
    如果你要選擇網頁中的所有元素
    *{
        一些設定
    }
